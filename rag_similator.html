RAG Learning Simulator

Understand RAG by watching it work.

RAG means Retrieval Augmented Generation. Instead of asking an AI model to answer from memory only, we first retrieve useful information from a knowledge base, add it to the prompt, then let the model generate a better grounded answer.

The simple idea

User Question
+ Relevant Knowledge
+ AI Model
= Better Answer

Your definition was close: a knowledge base is used to fetch the right information and enrich the prompt so the response is more accurate.
